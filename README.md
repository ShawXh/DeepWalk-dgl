# DeepWalk

- Paper link: [here](https://arxiv.org/pdf/1403.6652.pdf)
- Other implementation: [gensim](https://github.com/phanein/deepwalk), [deepwalk-c](https://github.com/xgfs/deepwalk-c)

## Dependencies
- PyTorch 1.0.1+

## How to run the code

Format of network:
```
1(node id) 2(node id)
1 3
...
```

To evalutate embedding on multi-label classification, please refer to [here](https://github.com/shawxh/evaluate_embedding)
